# Guia para Não Desenvolvedores

**Para quem é:** gestores públicos, cidadãos, jornalistas — qualquer pessoa que queira entender o projeto sem ser especialista em tecnologia.

**O que você vai aprender aqui:** o básico para navegar neste repositório e entender do que se trata a iniciativa.

---

## O que é este repositório?

Um repositório é como uma pasta compartilhada na internet, onde todo o código e a documentação de um projeto ficam guardados. Qualquer pessoa pode ver o que está sendo feito, sugerir mudanças e acompanhar o progresso.

## O que tem aqui dentro?

- **Documentação** (na pasta `Nuvem Brasil/`) — explica o projeto em português claro
- **Código** (em pastas como `apps/`, no futuro) — os programas que fazem o sistema funcionar
- **Cenários de teste** — descrições em linguagem simples de como o sistema deve se comportar

## Como ler a documentação

Comece pela pasta [`Nuvem Brasil/`](../README.md). Lá você encontra:

1. **Guia do Projeto** — o propósito e os objetivos da iniciativa
2. **Glossário** — explicação de termos técnicos
3. **Guias** — instruções para diferentes públicos

Se encontrar uma palavra estranha, confira o [Glossário](../GLOSSARIO.md).

## Como dar opinião ou sugestão

Você não precisa saber programar para contribuir. Pode:

- Abrir uma *issue* (um tópico de discussão) relatando um problema ou sugerindo melhoria
- Comentar em issues existentes
- Compartilhar o projeto com outras pessoas interessadas
