# Como Contribuir

**Para quem é:** qualquer pessoa — desenvolvedora ou não.

**O que você vai aprender aqui:** as formas de contribuir com a Nuvem Brasil.

---

## Para todos

### Reportar problemas ou sugerir melhorias

Abra uma *issue* no GitHub descrevendo:

1. O que você esperava que acontecesse
2. O que realmente aconteceu (se for um problema)
3. Passos para reproduzir (se aplicável)
4. Contexto adicional (capturas de tela, logs, etc.)

### Melhorar a documentação

A documentação está em `Nuvem Brasil/`. Se encontrar algo errado, confuso ou desatualizado:

1. Edite o arquivo diretamente (se estiver confortável com Git)
2. Ou abra uma issue apontando o problema

### Participar de discussões

Comente em issues e pull requests. Seu ponto de vista é bem-vindo.

## Pull requests

1. Descreva claramente o que o PR faz
2. Referencie a issue relacionada (se houver)
3. Mantenha PRs pequenos e focados
4. Certifique-se de que os checks de CI passam
